Title: Simon Task
Construct: cognitive control / response selection / spatial conflict

Purpose:
Show the participant-visible Simon trial timeline, emphasizing color-based responses while location is irrelevant.

Rows / conditions:
- Congruent: red-left or blue-right target; target side matches the response side.
- Incongruent: red-right or blue-left target; target side conflicts with the response side.

Timeline:
- Fixation: 500 ms; no scored response; centered +.
- Simon target: 1000 ms; press F for red and J for blue; colored circle appears left or right.
- Feedback: 500 ms; no response; correct, incorrect, or no response text.
- ITI: random 800-1200 ms; no response; blank screen.

Notes:
- Participants respond by color and ignore stimulus position.
- Red = F, blue = J.
- The configured condition tokens are red_left, red_right, blue_left, and blue_right.
